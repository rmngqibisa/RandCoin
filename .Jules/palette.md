# Palette's Journal

## 2024-05-20 - Visual Feedback in CLI
**Learning:** Even in a text-based interface (CLI), visual cues like emojis and formatted strings (e.g., currency) significantly reduce cognitive load. Users can instantly distinguish between success (✅) and error (❌) states without reading the full text.
**Action:** Always include visual status indicators in CLI outputs, not just plain text.
